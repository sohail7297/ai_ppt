# AI Presentation Studio

AI-powered presentation generator built with Streamlit, Gemini AI, and Python-PPTX.

## Features

* Generate presentations from topics
* Generate presentations from PDF/DOCX/TXT files
* Export to PowerPoint (PPTX)
* Export to PDF
* Multiple presentation themes
* Speaker notes generation
* Project history tracking
* Dashboard analytics

## Tech Stack

* Python
* Streamlit
* Gemini AI
* Python-PPTX
* Plotly
* ReportLab

## Installation

pip install -r requirements.txt

streamlit run app.py
